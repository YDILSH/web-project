Courses I'm Learning

Course 1: Networking Fundamentals

This course introduces the basic concepts of computer networking. Topics include network topologies, protocols, and devices like routers and switches. I'll learn how data is transferred across networks, understand the OSI model, and configure basic network settings. Key skills developed include setting up a home network, troubleshooting connectivity issues, and understanding network security fundamentals.

Course 2: Cyber Security Basics

This course covers the foundational principles of cybersecurity, focusing on protecting systems and networks from cyber threats. Topics include encryption, firewalls, malware, and risk management. I'll learn about common attack methods like phishing and denial-of-service attacks, and how to implement basic security measures. Key skills developed include securing personal devices, understanding security protocols, and identifying potential threats to systems.

Course 3: Introduction to Programming

This course provides an introduction to programming using Python. I will learn how to write basic programs, use variables and data types, and work with control structures like loops and conditionals. Topics also include functions, file I/O, and debugging. Key skills developed include problem-solving through coding, understanding algorithms, and gaining a foundation for learning more advanced programming concepts.
